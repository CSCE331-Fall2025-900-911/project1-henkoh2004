STEM Learning Platform

This was a group project done at the Aggie Coding Club. We developed a full-stack web application to teach STEM concepts with interactive content and UI designs. We had maintained PostgreSQL database with Prisma ORM to organize and query user data efficiently. By integrating OpenAI API to deliver adaptive study techniques, we improved engagement and effectiveness.

AI Chess Bot

I developed a machine learning chess engine in Python using PyTorch, trained on a dataset of 10M+ PGN games (from the LiChess database) to predict human-like moves. With a data pipeline using python-chess and Zstandard to stream compressed game files efficiently, I was able to generate training samples, and tested on the bots on Chess.com training samples.
